LEGACY SUPPORT

Supported Windows Versions by Version:

  • 2.0+: Windows 7, 8, 8.1, 10, 11
  • 1.0-1.9: Windows XP, Vista, 7, 8, 8.1

Architecture Support:

  • 2.0+: 32-bit and 64-bit systems
  • 1.0-1.9: 32-bit systems only

DEPRECATED FEATURES

Removed in Version 2.0:

  • Legacy driver installation method
  • Manual driver extraction requirement
  • Windows XP/Vista support
  • 32-bit only driver version
